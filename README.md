# Introduction to the Ethics and Regulation of AI: Course Materials

Welcome to the repository for the module **Introduction to the Ethics and Regulation of AI (E&RofAI)** in the M.Sc. Artificial Intelligence (MAI) programme at the Technical University of Applied Sciences Würzburg-Schweinfurt (THWS), Faculty of Computer Science and Business Information Systems.

This project uses a modern **single-source publishing** approach based on **Quarto**. From the Markdown source files (`.qmd`) it generates both interactive learning modules for Moodle and static PDF scripts. The entire course is in **English**.

## 🎓 Structure of the units

The repository is organised into 13 chronological units that follow the course of the semester. See [`_curriculum.md`](_curriculum.md) for the full unit list, topics and status.

## 🛠 Interactive elements (widgets)

A distinctive feature of this course are the **interactive HTML widgets** embedded directly in the scripts. They let students explore complex ethical trade-offs and legal mechanisms in a hands-on way. Widgets live in `/widgets`, organised by unit.

## 🚀 Technical stack

- **Framework:** [Quarto](https://quarto.org/)
- **Templates:** https://github.com/c-kraus — with many thanks to Prof. Dr. Christian Kraus!
- **Output formats:** `moodle-html` (via the THWS extension), `pdf`
- **Widgets:** Vanilla HTML5, CSS3 and JavaScript (Inter UI font stack)
